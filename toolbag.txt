Docker cmd's:
    # Log into the DB shell directly
    docker exec -it lycan_postgres psql -U jwolf -d lycan_db

PSQL cmd's (Run these inside the psql prompt):
    \dt          # List all tables (Check if 'vector_store' exists)
    \d+ <table_name> # Describe table structure (See columns/types)
    \q          # Quit psql

SQL cmd's (The "Lycan Audit" queries):
    # See where the mystery files came from
    SELECT DISTINCT metadata->>'source' FROM vector_store;

    # See how many "chunks" were created per file
    SELECT metadata->>'source', COUNT(*) FROM vector_store GROUP BY 1;

    # Check the total 'memory' size (row count)
    SELECT COUNT(*) FROM vector_store;

    # Peek at the actual text content
    SELECT content FROM vector_store LIMIT 5;
